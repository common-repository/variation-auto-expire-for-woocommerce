Change variation stock status to out of stock or delete on specific date-time (variation availability till specific date-time only).

== Description ==

This plugin for WordPress/WooCommerce store, by this plugin you can set variable product's variation availability to auto outofstock or delete on specific date-time.

This means your specific variation will available to purchase till provided specific date-time only. After date-time variation will auto out of stock or deleted (according to which you set action). Now custom timezones added : will be useful when local time and server time is different. Please check timezone settings once plugin activated for accurate time action.

== Installation ==

1. Upload "variation-auto-expire-for-woocommerce.zip" to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. OR, Go to WP admin panel, click 'Plugins' -> 'Add new'. In the search input box, type 'variation auto expire for woocommerce'.
4. Install and activate the plugin.
5. Plugin settings found beside activate/deactivate plugin on plugins page, also found on admin menu under woocommerce menu.
